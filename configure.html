Configure ModPorter w/Apache

The following are ModPorter's Apache configuration options suitable inside the application's VirtualHost directive.

Required Configuration

  • Porter <on|off>

    Default : off

    Enable or Disable ModPorter.

  • PorterSharedSecret <string>

    A secret phrase used for signing the parsed parameters' header containing the information about uploaded files. PorterSharedSecret is essential for ensuring a secure transaction between your application and ModPorter.

Optional Configuration

  • PorterMinSize <file size>

    Default : 0

    Possible Values :

    • 10M
    • 2048K
    • 1G

    If the total size of uploaded data is less than PorterMinSize, ModPorter will bypass parsing the multipart data.

  • PorterPermission <file permission hash>

    Default : 0x0666

    Possible Values :

    • 0x0006
    • 0x0777

    PorterPermission determines the default file permission of the temporary files created by ModPorter.
