ModPorter

Scale your file uploads

1. Install libapreq

ModPorter uses libapreq for parsing the multipart request.
  • Download libapreq 2.10 RC1 from here.
  • Install the C API : 
                
  % ./configure --with-apache2-apxs=/path/to/apache2/bin/apxs
  % make
  % make test
  % make install

2. Install ModPorter

  • Download the latest version of ModPorter from here.
  • Install using the rake command : 
                
  % rake
  • Configure ModPorter.

3. Install the Rails Plugin - acts_as_porter ( optional )

This step applies only if you wish to use ModPorter with a Rails application.
  • Install using the plugin using script/plugin inside your Rails application : 
                
  % script/plugin install git://acts_as_porter.git
